# vim-config-rhel-linux
simple vim config with CtrlP plugins (and js plugin for text editing)

## Installation 

git clone https://github.com/Nitesh058Raj/vim-config-rhel-linux.git

### After clonning this repo :

`chmod +x install.sh`

`./install.sh`
